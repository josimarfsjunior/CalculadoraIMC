# CALCULADORA DE IMC - ÍNDICE DE MASSA CORPORAL

Projeto criado com objetivo de construir uma calculadora de IMC - Índice de Massa Corporal.

## Regras Criadas:

### Campos Obrigatórios

1 - Peso em Kg: (Exemplo: 65);

2 - Altura em Mt: (Exemplo: 1.60);

### Função de Calcular IMC

1 - Ao ser acionada, executa a regra de calculo construída com base no 'Peso' e a 'Altura' informada pelo usuário.

2- Os campos e valores são limpados da tela a cada novo cálculo realizado pelo usuário.

3- Além de apresentar o resultado para o usuário é apresentado o IMC calculado em uma escala visual.
